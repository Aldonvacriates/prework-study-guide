HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element specifies the title of the webpage, which is shown in the browser's title bar or tab.
  • The meta element provides metadata about the HTML document, such as character set, author, and viewport settings.
  • The link element is used to link external resources, such as stylesheets, to the HTML document.
  • The script element is used to include or reference JavaScript code within the HTML document.
  • The div element is a container used to group and style block-level content.
  • The span element is an inline container used to group and style inline content.
  • The a element (anchor) creates hyperlinks to navigate between different pages or parts of the same page.
  • The img element embeds images into the webpage and requires the src attribute to specify the image source.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The border property sets the width, style, and color of an element's border.
  • The border-radius property creates rounded corners for an element's border.
  • The display property specifies the display behavior of an element (e.g., block, inline, flex).
  • The color property sets the color of text content inside an element.
  • The background-color property sets the background color of an element.
  • The font-family property specifies the font to be used for text content inside an element.
  • The font-size property sets the size of the text content inside an element.
  • The text-align property specifies the horizontal alignment of text inside an element.

Git

  • git status: checks what branch we are currently on and shows the status of changes.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add .: stages all changes in the working directory for the next commit.
  • git commit -m "commit message": commits the staged changes with a descriptive message.
  • git push: pushes the committed changes to the remote repository.
  • git pull: fetches and merges changes from the remote repository to the local repository.
  • git merge branch-name: merges the specified branch into the current branch.
  • git log: displays the commit history of the current branch.
  • git diff: shows the differences between the working directory and the staging area.
  • git clone repository-url: clones a remote repository to the local machine.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript can be used to make webpages interactive and dynamic.
  • A function is a block of code designed to perform a particular task when called.
  • Event listeners can be added to elements to execute code in response to user actions, such as clicks or key presses.
  • JavaScript objects are collections of key-value pairs that represent real-world entities.
  • JavaScript arrays are used to store multiple values in a single variable.
  • The document object represents the webpage and can be used to access and manipulate elements on the page.
  • JavaScript can be used to validate user input in forms before it is submitted to the server.
  • The console.log() method is used to print messages to the browser's console for debugging purposes.
  • JavaScript can be used to make asynchronous requests to a server using the Fetch API or XMLHttpRequest.
